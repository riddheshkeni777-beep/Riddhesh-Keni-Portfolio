## 🛠️ Local Development & Setup

To view and run this portfolio locally on your machine:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/riddheshkeni777-beep/Riddhesh-Keni-Portfolio.git
   cd Riddhesh-Keni-Portfolio
   ```
2. **Open the site**:
   - Simply open [riddhesh_portfolio.html](riddhesh_portfolio.html) in any modern web browser.
   - Alternatively, serve it using an editor extension like VS Code's **Live Server** to run it on a local port.
